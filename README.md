# ESP32-Flutter-Fall-Detection

## Introduction

This project is an IoT-based fall detection system using ESP32 and Flutter. The system detects falls, monitors heart rate and SpO₂, sends emergency notifications, and stores data on Firebase.

## Hardware

- ESP32
- MPU6050
- MAX30100
- OLED SSD1306
- Buzzer
- SOS Button

## Software

- Flutter
- Arduino IDE
- Firebase
- MQTT
- Telegram Bot
